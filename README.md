## "Полезные" советы для одинэсников!
1) Никогда не делай бэкапы, это занимает много времени да и место на диске не безграничное.
2) Чем больше запросов в цикле тем больше у тебя свободного времени (пока выполняется код)
3) Зачем тратить время на тестирование если за тебя все сделают пользователи. Ты один, а их много.
4) Получай реквизит через кучу точек. Так понятнее откуда он берется.
5) Ничего не логируй. Не оставляй следы.
6) Опаздывай на встречи - подождут.
7) Не пиши чистый код, тогда твои наработки не украдут конкуренты.
